# Fandom Wiki Scraping Assignment

## Website Chosen
I chose the Stardew Valley Wiki because it is my favorite video game and I was interested in how the wiki organizes farming information for players.

## What I Scraped
I scraped crop names and their associated growing seasons from the Crops page of the Stardew Valley Wiki.

## Why This Data Is Useful
This data is useful because it shows how the community structures farming information and helps players plan efficient strategies in the game.
